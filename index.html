⚙ Dashboard Info & Deployment Options
⭐ Option A — GitHub Pages Recommended
Free permanent URL. Access from any computer, any browser, anywhere — including restricted work networks. No installation required.

Steps:
  1. Go to github.com → Sign up (free)
  2. Create a new repository → name it khobar-dashboard
  3. Upload this HTML file, name it index.html
  4. Go to Settings → Pages → Source: main branch
  5. Your dashboard is live at https://YOUR-NAME.github.io/khobar-dashboard
  6. To update: replace the index.html file with new version
🔷 Option B — OneDrive / SharePoint
Best for corporate environments. Works within company network.
  1. Upload this HTML file to your OneDrive or SharePoint site
  2. Right-click → Share → Copy link
  3. Anyone in your organization can open directly in Edge/Chrome
  4. To update: replace the file with new version
☁ Option C — Google Drive
Upload to Google Drive and share via link.
  1. Upload this HTML file to Google Drive
  2. Right-click → Get shareable link → Anyone with link can view
  3. Share the link — opens directly in browser
  4. Note: Google may preview rather than run — use Download button to get the file if needed
💾 Option D — USB / Local File
Completely offline. No internet required. All data is embedded inside the file.
  1. Copy this HTML file to a USB drive
  2. On any computer: open USB drive → double-click the file
  3. Opens in Edge or Chrome automatically
  4. Works even on restricted computers with no internet
📥 How to Update This Dashboard
Option 1 — Upload new actuals: Use the "⬆ Upload 2026" button in the top-right to load a new .xlsx or .csv file. Data merges automatically.

Option 2 — Full rebuild: Send your latest Excel file + modification requests to get a completely rebuilt dashboard HTML file with new features and data.
